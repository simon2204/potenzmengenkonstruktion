Bedienungsanweisungen

Neue Zustände können per Doppelklick auf die Arbeitsfläche hinzugefügt werden.

Um Zustände zu ändern/löschen, muss ein Rechtsklick auf die Mitte des Zustands ausgeführt werden. Anschließend kann der Zustand in einem Dialogmenü verändert oder gelöscht werden.

Um Zustände zu verschieben muss ein Linksklick auf die Mitte des Zustands gehalten werden und zur gewünschten Position gezogen werden.

Um Zustände zu verbinden, muss erstmal mit einem Linksklick vom Rand des Ausgangszustands eine Linie zum gewünschten Zielzustand gezogen werden. Anschließend muss ein Eingabesymbol, Schreibsymbol und die Bewegung eingegeben und bestätigt werden.

Um Übergänge zu bearbeiten, muss ein Rechtsklick auf den jeweiligen Übergang ausgeführt werden. Anschließend kann der Übergang in einem Dialogmenü bearbeitet werden.

Es ist möglich, gleichzeitig einen Übergang und einen Zustand zu erstellen. Dafür muss mit einem Linksklick vom Rand des Ausgangszustandes eine Linie auf eine leere Stelle der Arbeitsfläche gezogen werden.
